## 📜 Introdução

A biblioteca **med-components** foi criada com o intuito de ser compartilhada entre as aplicações medgrupo de forma global.
Ela **não pode ser** incrementada com particularidades de outros projetos, apenas consumida.

As particularidades devem ser documentadas nos próprios projetos, seguindo o padrão presente nessa documentação. O time **[TheOffice](https://medgrupo.slack.com/archives/C013J7002G5)**
é responsável por **facilitar** e dar o **suporte** e **mentoria** necessários para criação dos mesmos. Com isso o conhecimento não fica centralizado e criamos uma cultura de documentação nos times.

## 🛠️ Instalação

No MSPRO, é necessario apenas:

- Criar uma branch a partir da Release-Branch ou da master
- Realizar npm install
- Garantir que no arquivo package.json existam as referencias abaixo:

```
"@ionic/core": "git+https://github.com/MEDGRUPOGIT/med-components.git#{versao_tag}",
```

Para instalar e utilizar a biblioteca em outra aplicação é necessario a instalação do fork que fizemos do ionic.

```
https://github.com/MEDGRUPOGIT/ionic-framework
```

## 💈 Utilização

### Medsoft

Para utilizar a biblioteca, no arquivo **./src/medsoft/global.scss**, importar:

``` scss
@import '~@ionic/core/med-components/med-components.scss';
```
Se existir a necessidade da utilização de **variáveis, functions e mixins SASS**, no arquivo em que eles serão
utilizados, importar:

``` scss
@import '~@ionic/core/med-components/abstracts';
```

### External Pages
