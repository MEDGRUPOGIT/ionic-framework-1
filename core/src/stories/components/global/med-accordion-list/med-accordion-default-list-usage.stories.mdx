# Accordion List

## Propriedades

| Propriedade     | Atributo         | Descrição                                    | Tipo                                                           | Default       |
| --------------- | ---------------- | -------------------------------------------- | -------------------------------------------------------------- | ------------- |
| **singleOpen**  | **single-open**  | Define a variação de estilo do componente.   | **boolean**                                                    | **true**      |
| **margin**      | **margin**       | Define a margin entre os itens do accordion. | **"xs"** \| **"sm"** \| **"md"** \| **"lg"** \| **undefined**  | **undefined** |

## Slots

| Slot         | Descrição                                                   |
| ------------ | ----------------------------------------------------------- |
| **default**  | Slot default destinado ao componente `med-accordion-item`.  |

## Default

```
<med-accordion-list>
  <med-accordion-item>...</med-accordion-item>
</med-accordion-list>
```

## Single Open

```
<med-accordion-list single-open="false">
  <med-accordion-item>...</med-accordion-item>
</med-accordion-list>
```

## Margin

```
<med-accordion-list margin="sm">
  <med-accordion-item>...</med-accordion-item>
</med-accordion-list>
```
