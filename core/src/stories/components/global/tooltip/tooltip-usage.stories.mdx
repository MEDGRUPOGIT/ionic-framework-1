# Tooltip

O tooltip utiliza o componente popover como base.

Documentação auxiliar ion-popover: **https://ionicframework.com/docs/api/popover**

**As propriedades marcadas com ⚠️ são extritamente necessárias.**

```
async presentPopover(ev: any) {
  const popover = await this.popoverController.create({
    component: 'med-tooltip', ⚠️
    cssClass: 'med-tooltip', ⚠️
    componentProps: { ... }, ⚠️
    mode: 'ios', ⚠️
    showBackdrop: false ⚠️
  });
  await popover.present();
}
```

## componentProps

```
componentProps: {
  header?: String,  // título do tolltip
  content?: String, // texto do tolltip
  buttonLeft?: {
    label?: String, // texto do botão
    icon?: String,  // nome do ícone
  },
  buttonRight?: {
    label?: String, // texto do botão
    icon?: String,  // nome do ícone
  },
},
```

## Simple

```
async presentPopover(ev: any) {
  const popover = await this.popoverController.create({
    component: 'med-tooltip',
    cssClass: 'med-tooltip',
    componentProps: {
      content: 'Este é um exemplo de tootip em parágrafo. Esse box contém o tamanho máximo de texto que pode ser exibido. Acima disso, utilize um modal.',
    },
    mode: 'ios',
    showBackdrop: false
  });
  await popover.present();
}
```
