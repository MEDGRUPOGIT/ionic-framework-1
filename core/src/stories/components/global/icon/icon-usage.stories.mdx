# Icon

## Propriedades

| Propriedade  | Atributo   | Descrição                | Tipo                                                                                              | Default       |
| ------------ | ---------- | ------------------------ | ------------------------------------------------------------------------------------------------- | ------------- |
| **color**    | **color**  | Define a cor do botão.   | **[Color](http://desenv.ordomederi.com/med-components/?path=/story/enums-enums--page#medcolors)** | **undefined** |
| **name**     | **name**   | Define o nome do icone.  | **"string"**                                                                                      | **undefined** |


Default
```
<ion-icon class="med-icon" name="med-arrow-left-circle"></ion-icon>
```

Colored
```
<ion-icon class="med-icon" name="med-arrow-left-circle" color="brand"></ion-icon>
```

Neutral color
```
<ion-icon class="med-icon" name="med-arrow-left-circle" color="neutral-5"></ion-icon>
```
