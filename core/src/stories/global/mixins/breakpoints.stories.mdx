# 🧪 Breakpoints

Mixins **globais** relacionados a **breakpoints**.

## 💈 Utilização

No arquivo em que a utilização dos **mixins** forem **necessárias**, importar:

``` scss
@import '~@ionic/core/med-components/abstracts';
```

Os breakpoints são quebras de layout padrões criadas com o intuito de padronizar as principais resoluções de viewport que as aplicações
do medgrupo necessitam.

<img src={breakpoints} alt="breakpoints" />

## 🧪 breakpoint-up()

### 💈 Utilização:

```
.class {
  @include breakpoint-up('md') { ... }
}
```

### Compila para:

```
@media (min-width: 768px) {
  .class { ... }
}
```

## 🧪 breakpoint-down()

### 💈 Utilização:

```
.class {
  @include breakpoint-down('lg') { ... }
}
```

### Compila para:

```
@media (max-width: 991.98px) {
  .class { ... }
}
```

## 🧪 breakpoint-between()

### 💈 Utilização:

```
.class {
  @include breakpoint-between('sm','lg') { ... }
}
```

### Compila para:

```
@media (min-width: 576px) and (max-width: 991.98px) {
  .class { ... }
}
```

## 🧪 breakpoint-only()

### 💈 Utilização:

```
.class {
  @include breakpoint-only('xxs') { ... }
}
```

### Compila para:

```
@media (min-width: 576px) and (max-width: 767.98px) {
  .class { ... }
}
```
